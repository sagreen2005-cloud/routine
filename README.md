# Shift Command Mobile

## Put it on GitHub

1. Create a new empty GitHub repository.
2. Unzip this package.
3. Upload every file and folder from inside the ZIP to the root of the repository.
4. Open repository **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select `main` and `/root`, then save.
7. Open the GitHub Pages link on your iPhone.
8. In Safari, tap **Share > Add to Home Screen**.

No npm, command line, or build process is required.

## Storage and backup

The app saves locally in IndexedDB and works offline after its first successful load.

For Google Drive backup on iPhone:
1. Open **Settings** in the app.
2. Tap **Share backup**.
3. Choose Google Drive in the iPhone Share Sheet.

## Current limitation

This GitHub Pages version supports a six-digit PIN. Face ID and dependable scheduled local notifications require a later native Capacitor wrapper built through Xcode.
